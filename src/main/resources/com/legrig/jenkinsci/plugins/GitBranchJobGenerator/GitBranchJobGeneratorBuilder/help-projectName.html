Set template for job name generation. Available variables:
  • TYPE - Build type from Rule
  • TYPE_CAP - Build type capitalized
  • BRANCH_FULL - Full Branch Name (including remote repo name]")
  • BRANCH_PROPER - Branch Name (excluding remote repo name]")
  • BRANCH - Massaged Branch Name (removing certain leading prefixes, like release/")
  • BRANCH_CLEAN - Clean version of ${BRANCH} - no special chatacters except for - and _
  • BRANCH_PROPER_CLEAN - Clean version of ${BRANCH_PROPER}
  • DAY- Current Day of Month as a number
  • MONTH- Current Month as a number
  • YEAR<- Current Year as a number/li>
  • DATE- Current Date as MONTH/DAY/YEAR
  • TIME - Current Time as HOUR:MINUTE:SECONDS
